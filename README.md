# html-svg-intermediate
A lightweight space for experimenting with HTML and SVG concepts, intended as a foundation for future projects and creative exploration.


This repository is a lightweight space for experimenting with HTML and SVG. It begins with simple shape explorations and gradually moves toward creative figures, such as a playful robot built entirely from basic SVG elements. The goal is to keep things flexible and open‑ended, serving as a foundation for future projects, experiments, and visual ideas.
